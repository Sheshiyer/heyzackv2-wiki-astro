# Short-Form Video Hooks

Ready-to-use hooks for TikTok, Instagram Reels, and YouTube Shorts.

---

## Platform Overview

| Platform | Optimal Length | Best Practices |
|----------|---------------|----------------|
| TikTok | 15-30s | Text overlay mandatory, 3-5 hashtags |
| Instagram Reels | 15-30s | 10-15 hashtags, sticker engagement |
| YouTube Shorts | 30-60s | Keyword-rich titles, #Shorts required |
| Facebook Reels | 30-60s | Storytelling captions, 5-10 hashtags |

### Posting Schedule

- **TikTok:** 6-9AM, 12-2PM, 7-11PM
- **Instagram Reels:** 9-11AM, 1-3PM, 7-9PM
- **YouTube Shorts:** 12-3PM, 5-9PM
- **Facebook Reels:** 1-4PM, 7-9PM
