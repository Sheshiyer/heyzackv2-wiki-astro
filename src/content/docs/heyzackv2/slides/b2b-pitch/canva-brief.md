# Canva Design Brief: HeyZack B2B Pitch Deck

## Project Overview

**Project**: Investor Pitch Deck
**Client**: HeyZack
**Slides**: 12
**Format**: 16:9 (1920x1080)
**Style**: Modern tech startup, premium, professional

---

## Brand Kit

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Primary Blue | #243984 | Headlines, data, trust elements |
| Primary Pink | #E82F89 | CTAs, accents, highlights |
| Black | #2E2D2C | Body text |
| Light Grey | #E9EDEF | Card backgrounds |
| White | #FFFFFF | Page backgrounds |
| Gradient | #243984 → #E82F89 | Title/closing slides, key moments |

### Typography
| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headlines | Poppins or Inter | Bold | 48-64px |
| Subheads | Poppins or Inter | SemiBold | 32-40px |
| Body | Inter | Regular | 18-24px |
| Data/Stats | Poppins | ExtraBold | 72-120px |
| Captions | Inter | Regular | 14-16px |

### Logo
- Primary: Stylized pink "Z" mark
- Wordmark: "HeyZack" in brand blue
- Placement: Top-left corner or centered on title/closing slides
- Clear space: Minimum 20px around logo

## Design Guidelines

### Layout
- **Margins**: 80px all sides
- **Grid**: 12-column for flexibility
- **Alignment**: Left-align body text, center headlines optionally

### Visual Hierarchy
1. Headline (largest, brand blue)
2. Key data (large numbers, pink accent)
3. Supporting text (regular weight, black)
4. Captions/labels (small, muted)

### Imagery
- Product photos: Clean, white background, professional
- Team photos: Professional headshots, consistent style
- Icons: Outline style, 2px stroke, brand colors
- Illustrations: Modern, minimal, tech-forward
