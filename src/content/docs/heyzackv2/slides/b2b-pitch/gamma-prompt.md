# Gamma.app Prompt: HeyZack B2B Pitch Deck

## Presentation Brief

Create a **12-slide investor pitch deck** for HeyZack, an AI-powered building automation startup seeking Series A funding.

**Audience**: Venture capital investors, strategic partners, property management executives
**Tone**: Professional, confident, data-driven
**Style**: Modern tech, clean, premium

## Theme & Styling

**Color Palette**:
- Primary: #243984 (deep blue)
- Accent: #E82F89 (vibrant pink)
- Background: #FFFFFF with subtle grey (#F8FAFC) sections
- Text: #2E2D2C (near black)

**Typography**:
- Headlines: Bold sans-serif (Inter or similar)
- Body: Regular weight, clean readability

**Visual Style**:
- Minimal, generous whitespace
- Gradient accents (blue to pink) for emphasis
- Outline icons, 2px stroke
- Clean data visualizations
