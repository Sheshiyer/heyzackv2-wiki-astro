# Canva Design Brief: HeyZack B2C Crowdfunding Deck

## Project Overview

**Project**: Crowdfunding Campaign Deck
**Client**: HeyZack
**Slides**: 12
**Format**: 16:9 (1920x1080)
**Style**: Lifestyle-focused, aspirational, modern tech

---

## Brand Kit

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Primary Blue | #243984 | Trust elements, security messaging |
| Primary Pink | #E82F89 | CTAs, excitement, emotion |
| Black | #2E2D2C | Body text |
| Light Grey | #E9EDEF | Card backgrounds |
| White | #FFFFFF | Page backgrounds |
| Gradient | #243984 → #E82F89 | Title/closing slides, key moments |

### Typography
| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headlines | Poppins or Inter | Bold | 48-64px |
| Subheads | Poppins or Inter | SemiBold | 32-40px |
| Body | Inter | Regular | 18-24px |
| Data/Stats | Poppins | ExtraBold | 72-120px |
| Captions | Inter | Regular | 14-16px |

### Logo
- Primary: Stylized pink "Z" mark
- Wordmark: "HeyZack" in brand colors
- Placement: Top-left corner or centered on title/closing slides
- Clear space: Minimum 20px around logo

## Design Guidelines

### Layout
- **Margins**: 80px all sides
- **Grid**: 12-column for flexibility
- **Alignment**: Center for headlines, left-align body text

### Visual Hierarchy
1. Headline (largest, brand blue)
2. Key value/emotion (large numbers, pink accent)
3. Supporting text (regular weight, black)
4. Captions/labels (small, muted)

### Imagery
- **Lifestyle**: Real people in real homes, not sterile showrooms
- **Product shots**: Clean, well-lit, consistent angles
- **Diversity**: Inclusive representation
- **Emotion**: Happy, relaxed, in control

### Tone
- **Approachable**: Not intimidating tech-speak
- **Aspirational**: The home you've always wanted
- **Trustworthy**: Real savings, real results
