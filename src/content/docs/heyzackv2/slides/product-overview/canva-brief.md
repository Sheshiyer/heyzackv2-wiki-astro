# Canva Design Brief: HeyZack Product Overview Deck

## Project Overview

**Project**: Product Overview Deck
**Client**: HeyZack
**Slides**: 10
**Format**: 16:9 (1920x1080)
**Style**: Clean, informative, product-focused

---

## Brand Kit

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Primary Blue | #243984 | Headlines, trust elements |
| Primary Pink | #E82F89 | CTAs, accents, highlights |
| Black | #2E2D2C | Body text |
| Light Grey | #E9EDEF | Card backgrounds |
| White | #FFFFFF | Page backgrounds |
| Gradient | #243984 → #E82F89 | Title/closing slides |

### Typography
| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headlines | Poppins or Inter | Bold | 48-64px |
| Subheads | Poppins or Inter | SemiBold | 32-40px |
| Body | Inter | Regular | 18-24px |
| Data/Stats | Poppins | ExtraBold | 72-120px |
| Captions | Inter | Regular | 14-16px |

### Logo
- Primary: Stylized pink "Z" mark
- Wordmark: "HeyZack" in brand blue
- Placement: Top-left corner or centered on title/closing slides
- Clear space: Minimum 20px around logo

## Design Guidelines

### Layout
- **Margins**: 80px all sides
- **Grid**: 12-column for flexibility
- **Alignment**: Left-align body, center headlines

### Product Photography
- Clean white backgrounds
- Consistent lighting and angles
- High resolution (min 1000x1000px)
- Show key features clearly

### Hierarchy
1. Category name (largest, brand blue)
2. Product count (accent, pink)
3. Product names (medium, black)
4. Descriptions (smaller, grey)
